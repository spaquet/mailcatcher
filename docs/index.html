Three Ways to Install

Docker (Recommended)
docker run -d \
  -p 1080:1080 \
  -p 1025:1025 \
  stpaquet/alpinemailcatcher
Ruby Gem
gem install mailcatcher-ng
mailcatcher
# View at http://127.0.0.1:1080
# Send to smtp://127.0.0.1:1025
From Source
git clone https://github.com/spaquet/mailcatcher
cd mailcatcher
bundle install && bundle exec mailcatcher

About MailCatcher NG

MailCatcher NG is an enhanced next-generation fork of the original MailCatcher by Samuel Cochran. We've rebuilt the web interface with modern design, added powerful new features like SMTP transcripts, email accessibility scoring, BIMI support, and seamless Claude integration.

The original MailCatcher remains the gold standard for email testing. MailCatcher NG builds on that solid foundation with a beautiful UI, next-generation features, and integration with modern development tools.

What's New

  • Modern, minimalist web interface
  • SMTP transcripts and server logs
  • Email accessibility scoring
  • BIMI brand indicator support
  • Claude Plugin & MCP Server
  • Advanced email authentication display
  • Full UTF-8 and internationalization
  • Persistent message storage
