MailCatcher NG REST API

Base URL: http://127.0.0.1:1080

Message Retrieval

Get and retrieve email messages

GET
/messages
List all caught messages

Parameters

Name Type Description
qstringSearch term (optional)
sortstringasc|desc (default: desc)

Example

curl http://127.0.0.1:1080/messages
GET
/messages/:id.json
Get message details with metadata

Example

curl http://127.0.0.1:1080/messages/1.json
GET
/messages/:id.html
Get HTML version of message

Example

curl http://127.0.0.1:1080/messages/1.html
GET
/messages/:id.plain
Get plain text version of message

Example

curl http://127.0.0.1:1080/messages/1.plain
GET
/messages/:id.source
Get raw SMTP source with headers

Example

curl http://127.0.0.1:1080/messages/1.source
GET
/messages/:id.eml
Download message as EML file

Example

/messages/:id/parts/:cid

Example

GET
/messages/:id/transcript.json

Example

/messages/:id/links.json

Search

Advanced message search

GET
/messages/search
Search with filters

Parameters

Name Type Description
qstringSearch term (required)
has_attachmentsbooleanFilter by attachments (optional)
fromdateFrom date ISO 8601 (optional)
todateTo date ISO 8601 (optional)

Example

curl 'http://127.0.0.1:1080/messages/search?q=invoice&from=2025-01-01T00:00:00Z'

Management

Delete and manage messages

DELETE
/messages/:id
Delete single message

Example

curl -X DELETE http://127.0.0.1:1080/messages/1
DELETE
/messages
Delete all messages

Example

curl -X DELETE http://127.0.0.1:1080/messages
POST
/messages/:id/forward
Forward message to another address

Example

curl -X POST http://127.0.0.1:1080/messages/1/forward \
  -H 'Content-Type: application/json' \
  -d '{"recipient":"user@example.com"}'
DELETE
/
Shutdown server gracefully

Example

curl -X DELETE http://127.0.0.1:1080/

Real-time Updates

WebSocket for live notifications

WS
ws://127.0.0.1:1080/messages
Real-time message updates

Events

Event Description
addNew message arrived
removeMessage deleted
clearAll messages cleared
quitServer shutting down

Example

const ws = new WebSocket('ws://127.0.0.1:1080/messages');
ws.onmessage = e => console.log(JSON.parse(e.data));
